### Font Family
- Tipo de fonte de um elemento
- Lista de fontes e ordem de prioridades
- inclui fallback fonts

p{
    font-family: 'Times New Roman', Times, serif;
}

caminhho ideal -> Times New Roman
fallback -> Times -> serif
- serif -> "dobrinhas"
- sans -> "sem"
